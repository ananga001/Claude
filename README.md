# ClaudeSol001

A Visual Studio solution built with .NET 9, used for exploring agentic AI coding with Claude Code.

## Projects

| Project | Type | Description |
|---|---|---|
| [ConsoleApp1](./ConsoleApp1/README.md) | Console App | Simple .NET 9 console application |
| [HelloWorldWinForms](./HelloWorldWinForms/README.md) | WinForms App | Windows Forms app with a Hello World form |
| CalculatorMVC | ASP.NET Core MVC | Calculator, Loan Approval (Chain of Responsibility), User Management, Cookie Auth |

## Requirements

- .NET 9 SDK
- Windows (required for WinForms project)
- Visual Studio 2022 v17+
