# HelloWorldWinForms

A .NET 9 Windows Forms application with a single form displaying "Hello, World!".

## Run

```powershell
dotnet run --project HelloWorldWinForms\HelloWorldWinForms.csproj
```

Or open the solution in Visual Studio, set `HelloWorldWinForms` as the startup project, and press **F5**.

## Target Framework

- `net9.0-windows`

## Form

| File | Purpose |
|---|---|
| `Form1.cs` | Form logic |
| `Form1.Designer.cs` | Auto-generated layout (Label: "Hello, World!") |
| `Program.cs` | Entry point |
